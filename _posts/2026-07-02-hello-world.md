This is a sample blog post. You can write in **Markdown** and Jekyll will render it to HTML.

## How to add new posts

1. Create a new file in `_posts/` with the format `YYYY-MM-DD-title.md`
2. Add front matter at the top:

```yaml
---
layout: post
title: "Your Post Title"
date: 2026-07-02
description: "A short description for the listing page."
nav: blog
---
```

3. Write your content in Markdown below the front matter.
4. Push to GitHub and it builds automatically.

## Features you get for free

- **Markdown rendering** — write naturally, no HTML needed
- **Code blocks** with syntax highlighting
- **Auto-listing** on the blog index page
- **Dates and permalinks** handled by Jekyll

That's it. Happy writing.
